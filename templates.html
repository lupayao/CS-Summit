Tips for Using These Resources

Customize for Your Classroom

All templates are provided in editable formats. Modify language, requirements, and complexity to match your students' experience levels.

Set Up GitHub Classroom

These resources work best when used with GitHub Classroom. Create an organization for your class to easily distribute assignments.

Plan for Onboarding

Allow extra time at the beginning of your course for students to get comfortable with Git. Use the intro materials before diving into complex projects.

Peer Learning

Encourage students to help each other with Git operations. Consider a "Git expert" role that rotates among team members during group projects.
